# Pollination Honeybee vtk plugin

The honeybee-vtk plugin adds hbjson visualization functions to Pollination.

## Sources

- `honeybee-vtk` Python package. [PyPI](https://pypi.org/project/honeybee-vtk/), [source](https://github.com/ladybug-tools/honeybee-vtk)
- `vtk` Python package. [PyPI](https://pypi.org/project/vtk/), [source](https://github.com/Kitware/VTK)
